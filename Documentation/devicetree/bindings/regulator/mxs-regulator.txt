Regulators for Freescale MX233/MX28

Required properties:
- compatible:
  - "fsl,mxs-regulator-vddd" for VDDD
  - "fsl,mxs-regulator-vdda" for VDDA
  - "fsl,mxs-regulator-vddio" for VDDIO
- reg: Address and length of the register set for the device. It contains
  the information of registers in the same order as described by reg-names
- reg-names: Should contain the reg names
  - "base-address"	- contains base address of regulator
  - "status-address"	- contains status address of regulator

Any regulator property defined as part of the core regulator
binding, defined in regulator.txt, can also be used.

Example:

	reg_vddd: regulator@80044040 {
		reg = <0x80044040 0x10>, <0x800440c0 0x10>;
		reg-names = "base-address", "status-address";
		compatible = "fsl,mxs-regulator-vddd";
		regulator-name = "vddd";
		regulator-min-microvolt = <800000>;
		regulator-max-microvolt = <1575000>;
		regulator-boot-on;
		vddd-supply = <&reg_vdda>;
	};
